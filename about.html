About SteveBayerIN

Steve Bayer was first interested in the stock market at the age of 13. He also developed an interest in websites when he saw the promotion for the movie 'Independence Day' on Yahoo in 1996. In 2007, after generating some funds connected to an internet banking account while working in a call center, he decided to rent a web server and start selling website services.

Along the way, he has learned how to set up a shared server, install Wordpress, install Drupal, wireframe websites and applications, theme Wordpress, theme Drupal and buy the NIFTYBees.
