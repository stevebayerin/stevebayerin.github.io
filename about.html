About SteveBayerIN

Steve Bayer was first interested in websites when he saw the promotion for the movie 'Independence Day' on Yahoo in 1996. In 2007, after generating some funds connected to an internet banking account while working in a call center, he decided to rent a web server and start selling website services.

Along the way, he has learned how to set up a shared server, install Wordpress, install Drupal, wireframe websites and applications, theme Wordpress, theme Drupal and invest on the BSE in India while outperforming the SENSEX and NIFTY by about 10 percent since 12th August 2015.
